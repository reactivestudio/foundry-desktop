# Конспекты советов бюро

Рабочий материал дизайн-справочника foundry-desktop: подборки советов
Артёма Горбунова и Ильи Бирмана, законспектированные по статьям
с bureau.ru — тезисы с дословными формулировками, картинки из советов
(в `img/`), примеры-кейсы и заготовки демо «плохо/хорошо/лучше»
для страниц примеров.

Страницы живых примеров: группа 01 «Дизайн продуктов» —
`design/bureau-product-design.html`; группа 02 «Типографика и вёрстка» —
`design/bureau-typography.html`; базовый чек-лист вёрстки и UI —
`design/bureau-checklist.html`.
