## Топ-8 свежих приёмов 2026 → как применяем

1. **Dark-first, не тоггл** (Linear/Vercel) → токены от тёмной базы (глубокий сине-чёрный, не #000). ✅ уже так.
2. **Слои поверхностей вместо чёрного** → elevation светлотой + 1px бордер, не коробки-рамки. ✅ канон.
3. **Dark glassmorphism с амбиентным свечением** → орб как источник ambient-света, стекло ловит его по краям. ⏳ усилить.
4. **4-уровневая система приоритетов цветом** (APEX) → статусы кодировать критично/активно/нейтрально/приглушено, не радугой. ✅ семантика.
5. **Плотность типографикой, не сжатием** → приглушённая мета, акцент на значении, mono для чисел/хешей/путей. ✅ tabular+mono.
6. **AI-агент как встроенная панель** (IDE-агенты 2026) → орб-беседа док-панелью рядом с диффом/канбаном, общий контекст. ⏳ бэклог.
7. **Пайплайн как граф связанных нодов** → CRISPY-вью нодами со статус-точками и прогрессом, не плоским логом. ⏳ бэклог.
8. **Микровзаимодействия на статусах/наведении** → hover-подъём карточек, пульс live, переходы 150–200 мс ease-out. ✅ motion-токены.

## Проход 2 (мой Behance-ресёрч под 4 паттерна) — верифицировано глазами

Оригиналы скачаны в `behance/b2-*`. Все проекты 2025–2026.

**Flow / DAG (FlexiFo `b2-orig_wfbuilder.png`, Aivora `b2-orig_aivora.png`). ✅ построено (§9).**
Узел = нейтральная карточка + **цветной icon-chip слева** (тип кодирует иконка, не заливка
всей карточки); рёбра пунктиром; подписи ветвления `True/False` в мелких пилюлях на ребре;
терминаторы Start/End; правый инспектор Settings/History/Preview. → У нас: узел на `#14121E`,
тип — icon-chip ультрамарином, статус прогона отдельным маркером **цвет+форма** (success кружок
#4ADE80 / error треугольник #F87171 / pending пульс-кольцо #FBBF24); активный путь — ультрамарин
+ слабый glow; числа tabular.

**Health scorecard (Vantix `b2-orig_vantix.png`, CONNECTIS `b2-orig_connectis.png`).**
Строка = `лейбл + сегмент-бар на тёмном треке + число справа В ЦВЕТ статуса` (заливка и число
одного цвета = двойное кодирование). CONNECTIS: одна лента с сегментами normal/warning/critical +
строка-легенда счётчиков с точками; статус-чипы фильтра со счётчиком-бейджем. → **Подтверждает
наш §4 scorecard**; апгрейд — красить число в цвет статуса и добавить одну сегмент-ленту с легендой.

**Version/run timeline (Astra `b2-astra_top.jpg`, Aivora).**
Ось-колонки недель с **чередующейся штриховой подложкой диапазонов** (зебра дат — плотность без
линеек); стадии каскадом пилюль-баров со сдвигом + violet-свечение активных; узлы-вехи со связями
и hover-пилюлей Compare/Details. → У нас: к таймлайну прогонов добавить штриховую подложку
диапазонов дат; обводка узла = итог прогона.

**Density/appearance settings (Untitled UI, не верифиц. — Dribbble антибот). ✅ построено (§10).**
Тема и плотность карточками-**мини-превью окна** с радио-выделением активной ультрамарином. → У нас:
пресеты Compact/Cozy/Spacious мини-превью с реальным шагом отступов; темы — миниатюра нашего стека
поверхностей #07060B→#0D0C14→#14121E с орбом.
