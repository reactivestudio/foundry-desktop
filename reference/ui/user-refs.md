# Референсы от пользователя (разбор приёмов)

Пользователь присылает референсы картинками в чат. Оригинальные файлы недоступны
(вставка в чат не даёт URL/байтов), поэтому здесь — разбор с извлечёнными
приёмами для применения к макету foundry-desktop. Помечаю: ✅ уже есть у нас,
⏳ стоит применить, ✖ не наш кейс (single-user / светлая тема).

Партия 1 (5 шт). Ждём ещё.

---

## R1 — Attio «Enhance your teamwork» (тёмный CRM)

- ⏳ **Инлайн-редактирование ячейки с цветной обводкой выбора** (In Trial, 10–24…):
  выделенная/редактируемая ячейка — рамка акцентом, разные цвета под разных
  соредакторов. → У нас: инлайн-правка полей change'а (имя ветки, ревьюер) прямо
  в строке инбокса, без модалки.
- ✖ **Мультиплеер-присутствие: цветные ярлыки-имена на ячейках/строках** (Anna,
  Andrew, James) + курсоры соредакторов. → Мы single-user; отложить, но механика
  «кто смотрит» пригодится, если появятся команды.
- ⏳ **Пер-строчный дропдаун прав** (No Access / Full Access / Read Access): право
  меняется прямо в строке списка. → Паттерн «действие-select в строке» ложится на
  наш «сменить стадию/ревьюера» инлайн.
- ⏳ **Богатый тулбар комментария** (Text ∕ Highlight ∕ B I U ∕ список) на плавающей
  панели заметки. → Тулбар типизированного комментария в диффе: подсветка кода,
  цитата строки, список.
- ✅ **Статус = точка + текст** в ячейке Stage (Demo Scheduled ●, In Trial ●).
- ⏳ **Заголовки колонок с мелкой иконкой типа поля** (Stage ▮, Team Size ◔, Date ▦).
  → В таблицах аналитики/холдингов — иконка типа колонки в шапке.

## R2 — Spark (тёмная почта на градиентных обоях)

- ⏳ **Свайп-экшены из-под строки** (SNOOZE янтарным из-под письма). → Свайп по
  строке инбокса: отложить/архивировать change без открытия.
- ✅ **Выбранный элемент = яркая заливка** (письмо залито синим). ✅ **применено в §16**:
  карточка выбранного агента залита ультрамарином 16% с обводкой 75% — выбор виден
  с другого конца стола, а не только рамкой.
- ⏳ **Ряд тегов/лейблов на детали** (Inbox ×, Private ▾ янтарный) — снимаемые
  чипы-теги. → Метки change'а (приоритет, проект) чипами в шапке инспектора.
- ⏳ **Тред-переписка пузырями внутри детали** (свои — синие, чужие — серые, с
  аватарами). → История ревью-обсуждения в инспекторе пузырями (агент ↔ ты).
- ✅ **Сайдбар с цветными точками аккаунтов + счётчик-пилюля** (Work ● 25 зелёным).
- ⏳ **Группы в списке с тихими заголовками** (Notifications / Newsletters) и
  action-иконкой справа у группы. → Заголовки групп инбокса могут нести
  групповое действие (свернуть/очистить).

## R3 — Spark Redesign hero (окно на обоях Big Sur)

- ✅ **Окно приложения лежит на цветных обоях** — ровно наша презентационная
  подача (аврора-подложка). Подтверждает направление, новых приёмов нет.

## R4 — Chainview (тёмный DeFi-дашборд) — самый ценный для нас

- ⏳ **KPI-полоса сверху**: карточки (DeFi Deployed $598,220 · +3.8% 24h · 22
  positions), у каждой иконка+лейбл, крупное число, дельта, микроконтекст. →
  Прямо усилить нашу аналитику §4: KPI как верхняя полоса, а не только сетка.
- ⏳ **Порог-аннотация у метрики риска**: Health (lowest) 1.34 ⚠ «Aave v3 liq at
  1.05». → У «зависших лупов»/SLA — показывать порог рядом со значением.
- ⏳ **Hero area-график с crosshair-тултипом, вертикальной линией и точкой-
  эндпоинтом** (Total net worth, тултип «+$28,900»). → Апгрейд графика аналитики:
  вертикальный курсор + значение в точке наведения (у нас пока голые спарклайны).
- ⏳ **Инлайн-бар ALLOC прямо в ячейке таблицы** (Holdings: …| 31.4% ▬▬▬). →
  В таблице проходимости/размеров change'ей — мини-бар доли в самой ячейке.
- ⏳ **Донат-аллокация с подписями-процентами на сегментах** + легенда снизу
  (Arbitrum/Base/Solana со значениями). → Распределение change'ей по проектам/
  стадиям донатом (осторожно: Тафти не любит пироги — только если ≤4 долей).
- ✅ **Цианом кодируется активное/live**, count-бейджи в навигации (DeFi 4, Alerts 2).
- ⏳ **Промо/CTA-карточка внизу сайдбара** (Chainview Desk + кнопка). → Низ нашего
  сайдбара: статус агента как карточка, либо «запустить луп».
- ✅ **Числа в таблице вправо, tabular; дельты 24h зелёным/красным.**

## R5 — fxpro-подобный лендинг «All top markets» (СВЕТЛАЯ тема)

- ⏳ **Dot-matrix микрографик**: спарклайн из точек, цвет кодирует направление
  (зелёный рост / красный падение), подкрашены только значимые точки. → Наш
  «активность лупов» точечной матрицей вместо линии; цвет — только на значимом
  (совпадает с приёмом fxpro, что уже в каноне).
- ⏳ **% изменения как цветная пилюля** (0.1% зелёная / 1.1% красная) рядом с ценой.
  → Дельты метрик пилюлей, а не только стрелкой.
- ⏳ **Сетка карточек-инструментов** (флаг + имя + микрографик + %/цена + Trade). →
  Раскладка «карточка проекта/стадии» на обзорном экране портфеля change'ей.
- ✖ Тема светлая — берём паттерн, не палитру.

## Партия 2 (~30 шт) — сгруппировано по приёмам

Оригиналы недоступны (те же байты чата). Помечаю ✅ есть / ⏳ применить / ✖ не наш кейс.

### Многоколоночные dev-раскладки
- ⏳ **4-панельный плейграунд: Code · Actions · Console · Storage** (Okashi/Soroban).
  Консоль с маркерами ввода/вывода `»`/`«`; Storage — инспектор key-value карточками.
  → Референс мульти-пейна: дифф · действия стадии · лог · инспектор артефакта.
- ⏳ **Верхние вкладки-режимы + узкий левый icon-rail + Compile ⌘K внизу** (Okashi).
  → Инбокс/Канбан/Лог вкладками, rail инструментов, «Запустить ⌘⏎» в углу.

### Пайплайн и здоровье — самое ценное (NextGen/AppDynamics)
- ⏳ **Flow-map (санки) с лентами success/error/pending** зел/красн/серый между узлами.
  → CRISPY как поток пропускной способности: где течёт, где падает. Новое к канбану;
  ложится на «граф нодов» (Behance-8 #7) и «параллельную ленту» Worldchess.
- ⏳ **Сегментные бары здоровья** «2 critical / 0 warning / 42 normal» + **scorecard-строка**
  лейбл+цветной бар+счётчик (Normal 73.9k / Slow 390 / Errors 1.3k). → Здоровье стадий/лупов,
  плотно, цвет=форма.
- ⏳ **dot-scatter мини-графики ошибок по времени** (gauge — с осторожностью, Тафти против).
- ⏳ **Троабл-быстрокарточки в сайдбаре** (Slow / Errors под Troubleshoot). → «Зависшие /
  Упавший CI» быстрофильтрами (= виджет-сводка Бирмана).

### История версий / прогонов (Notion-automation)
- ⏳ **Таймлайн-скаттер версий** (точки время×день, зелёная=значимая) + **карточка версии**
  «Version 53 · дата · описание · аватары · ✓ · Compare/Details» + пейджер + диапазон «30 Days».
  → История прогонов лупа / версий change'а с Compare/Details.

### Командная палитра / поиск (SocAI, Search states, Shipr)
- ⏳ **Палитра с секциями ACTIONS / RECENT / QUICK ACTIONS**, у пункта **буква-хоткей** (M/P/S)
  и тип, футер-легенда «↑↓ navigate · ↵ select · esc close». → Апгрейд §8 ⌘K.
- ⏳ **Live-фильтр с Clear и hover-подсветкой строки**; **suggestion-чипы под поиском**.

### KPI / метрики (Quick Stats, Cashflow)
- ⏳ **KPI-карточка: иконка + ↗18% дельта + крупное число + «Prev: $210,450»**. → Шаблон §4,
  исторический базлайн строкой (усиливает R4).
- ⏳ **Аннотация на линии** (точка «22 Oct · $138,400») + **пунктирный target-базлайн**.
- ⏳ **Инлайн AI-подсказка в дашборде** «Проанализировать, почему …?» + инпут.
- ✖ Многосерийная «волна» (crypto) — декоративно; берём только **series-чипы + гистограмму-скраббер**.

### Пропорции / микрографики в карточках (Activity Summary, Whoop, Market)
- ⏳ **Пропорц-бар тик-сегментами с инлайн-подписями** WORK 58% / ABSENCE 28% / OTHER 14%.
  → «Доля change'ей по стадиям» — плотнее и честнее пирога.
- ⏳ **Мини-бар-чарты в стат-картах с осью дней** (стек-бар / range+линия / сегмент-бар).
- ✅ **Строка списка с инлайн-спарклайном + направленный %** — усиливает R5.

### Канбан / список (Synapse, Project UI/UX)
- ⏳ **Приоритет strength-иконкой + прогресс-кольцо % в строке + сортируемые колонки (⇅)**.
- ✅ **Анатомия карты**: eyebrow-категория · теги · футер-метрики. ✅ **применено к карточкам
  агента §16** (eyebrow «CLI-агент · Anthropic» · теги-возможности · футер-метрики фактом);
  заголовок колонки = точка + счётчик — ✅ в §6. Футер-метрики на картах §6 — ещё ⏳.

### Настройки — наш новый экран (Untitled UI, Front Door, App Settings)
- ⏳ **Пикер темы мини-превью-окнами + пресеты плотности Default/Compact/Spacious мини-превью**
  + Save/Cancel. → Плотность реально нужна десктопному тулу.
- ⏳ **Тоггл-пилюли состояний** (вкл=зелёная заливка) + **выбор акцента свотчами**;
  **хоткей-чипы** — ✅ применены на карточках агента §16 (клавиши 1–5).

### Композер / ввод (Brainwave, Cursor, VidGen)
- ⏳ **Ghost-text автодополнение в поле** (серым продолжение). → Инпут комментария/промпта агенту.
- ⏳ **Плюс-меню вложений + drop-zone «Drop anything here» + @-mention чип + пикер модели +
  состояние записи**. → Прикрепить патч/лог в change; @-упоминание change'а в комменте.
- ⏳ **Param-чипы у стеклянной AI-панели** (9:16·1080p·15s·Help). → Плавашка агента с чипами запуска.

### Прочее
- ✅ **Пустое состояние: доменная иллюстрация + primary CTA** (Okashi «No Projects yet»).
- ⏳ **Route/progress-линия с узлом-серединой** (SJC •••✈••• DXB) + **амбер highlight-strip**. →
  Мини-таймлайн стадии строкой; выделение ключевой строки заливкой (усиление выбора, как Spark).
- ✖ Cream-bento и мобильные aurora-карточки — атмосфера/паттерн, не наша палитра.

## Мастер-список кандидатов (обе партии, по убыванию ценности)

**Аналитика §4**
1. KPI-карточка: иконка + число + цветная дельта + «Пред.: …» (Quick Stats, R4).
2. Hero-график: area + crosshair-тултип + эндпоинт + аннотация точки + target-базлайн (R4, Cashflow).
3. Пропорц-бар тик-сегментами «доля change'ей по стадиям» (Activity Summary).
4. Инлайн-бар доли в ячейке таблицы (R4).

**Пайплайн / здоровье — новый блок**
5. Flow-map стадий success/error/pending лентами (NextGen).
6. Сегментные бары здоровья + scorecard-строки (NextGen).
7. История прогонов: таймлайн-скаттер + карточка версии Compare/Details (Notion-auto).

**Главное окно / инбокс**
8. Приоритет strength-иконкой + прогресс-кольцо + сортируемые колонки (Synapse).
9. Порог-аннотация у метрики риска (R4).
10. Свайп-экшены и снимаемые теги; выделение ключевой строки заливкой (Spark R2).
11. Инлайн-правка полей change'а в строке (Attio R1).

**Ревью / дифф**
12. Тред-обсуждение пузырями агент↔человек в инспекторе (Spark R2).
13. Тулбар типизированного комментария (Attio R1).
14. Композер: ghost-text, @-mention, drop-zone вложений, пикер модели (Cursor/Brainwave).

**Оверлеи / навигация**
15. Палитра ⌘K с секциями + побуквенные хоткеи + легенда (SocAI).
16. Console `»/«` + Storage KV-инспектор как панели (Okashi).

**Настройки — новый экран**
17. Пикер темы и пресеты плотности мини-превью-окнами (Untitled UI).
18. Тоггл-пилюли состояний + акцент свотчами + хоткей-чипы (Front Door, App Settings).

**Микро**
19. Dot-matrix / мини-бары активности лупов по дням на значимом цвете (R5, Whoop).
20. Ruler/tick-слайдер с target-маркером для числовых настроек (Padel).
21. Анатомия канбан-карты: eyebrow + теги + футер-метрики (Project UI/UX) — апгрейд §6.
